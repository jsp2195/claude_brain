# Graph Autoencoder for Latent Compression of PBX Microstructures

## Overview

Graph autoencoder project focused on learning compact latent representations of PBX microstructure. Encode the topology and geometry of heterogeneous particle assemblies into a latent space useful for downstream analysis, generation, and potentially microstructure-to-response modeling.

## Status

Active project definition and technical framing, especially in the context of the AML26 summer student effort.

## Key Goals

- Define the graph representation cleanly
- Build a robust latent compression pipeline
- Establish meaningful reconstruction and latent quality metrics
- Scope the work so it is executable by a student while still contributing to the broader research program

## Deadlines

Tied to AML26 summer student timeline — orientation late May 2026, on-site work beginning early June 2026.
