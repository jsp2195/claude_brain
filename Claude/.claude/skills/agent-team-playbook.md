# Skill: Agent Team Playbook

Reusable workflow for disciplined coding tasks.

## Objective
Deliver correct minimal diffs with independent verification and explicit ownership.

## Workflow
1. Intake + classify task risk.
2. Route smallest viable team.
3. Lock file ownership.
4. Implement/debug in scoped stages.
5. Verify independently.
6. Report evidence and residual risks.

## Output contract
- Ownership map
- Compact handoffs
- Verification evidence
- Final summary with confidence level
