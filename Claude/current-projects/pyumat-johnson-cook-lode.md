# PyUMAT Johnson-Cook with Lode Angle

## Overview

Constitutive modeling project involving a PyUMAT implementation of Johnson-Cook-type plasticity extended or modified to account for Lode angle effects. Capture more realistic dependence of material response on stress-state geometry.

## Status

Active technical work.

## Key Goals

- Get the constitutive formulation implemented correctly
- Verify the stress update behavior
- Ensure the Lode-angle dependence behaves as intended
- Validate the model against expected constitutive behavior or reference cases

## Deadlines

No fixed external deadline. Part of active mechanics modeling work.
