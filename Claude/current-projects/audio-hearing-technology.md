# Audio Engineering / Hearing Technology Project

## Overview

Research and engineering workstream around audio, hearing augmentation, and machine learning. Signal processing, inference, and intelligent systems for hearing-related tasks rather than conventional materials modeling.

## Status

Active exploratory development.

## Key Goals

- Define the MVP clearly
- Determine the core technical contribution
- Build a coherent prototype rather than a loose concept
- Establish whether the main contribution is inference, control, scene modeling, or adaptive hearing assistance

## Deadlines

No fixed external deadline. Active exploratory parallel project.
