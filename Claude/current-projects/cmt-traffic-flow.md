# CMT Multi-Agent Traffic Flow Project

## Overview

Multi-agent modeling of traffic flow using structured CMT traffic data. Use agent-based or multi-agent methods to understand, simulate, or predict traffic behavior in a principled way.

## Status

Early-stage development. Data access, data extraction, and technical setup are part of the current work.

## Key Goals

- Pull and organize the relevant CMT data
- Define the multi-agent modeling setup clearly
- Build the first working analysis or simulation pipeline
- Determine what outputs or predictive tasks matter most
