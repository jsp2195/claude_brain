Never use em-dash. Always use en-dash (–) in all outputs.

**Why:** Em-dashes are a clear marker of AI/LLM-generated text. The goal is for outputs to sound like Jarett, not like a language model.

**How to apply:** Every dash that would normally be an em-dash should be an en-dash instead. This applies to all contexts – prose, code comments, proposals, emails, notes, everything. No exceptions.
